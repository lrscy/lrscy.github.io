前言

本文是在Ubuntu17.04系统上安装NVIDIA驱动。

在Ubuntu16.04 LTS上安装驱动时是从NVIDIA 官网上下载.run文件然后按照提示安装，但是在Ubuntu17.04上并不成功。因此寻找到了一种更为简便的方法，记录在此。
